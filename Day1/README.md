#Gemini CLI Chatbot

A simple, interactive command-line interface (CLI) chatbot built with Python and Google's Gemini API (gemini-1.5-flash).

*Features*
- Terminal Chat Interface: Simple loop allowing for a conversation

- Gemini Integration: uses gemini-1.5-flash

- Error handling: handles missing API keys and runtime exceptions

*Prerequisites
- Python 3.8+
- Google Gemini API Key (Get one from Google AI Studio; must have a 18+ account)

*Installation & Setup
- Clone or download this repository to your local machine
(In one bootcamp folder with a virtual environment created)
git clone <repository-url>

- Install the required package:

Bash
pip install google-generativeai
